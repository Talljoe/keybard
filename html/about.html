KeyBard, the Svalboard Keymap Editor

Developing for html/css - Make it super fast to reload the page so you can see changes w/o reconnecting your keyboard each time.

  1. Enable 'Record' in menu.
  2. Launch, and select your keyboard.
  3. Enable 'Playback' in menu.
  4. Reload page - It should now be near instant.
  5. (Note: Keyboard changes will not be performed at this point.)
